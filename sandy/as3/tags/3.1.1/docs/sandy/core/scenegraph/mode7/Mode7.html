Packagesandy.core.scenegraph.mode7
Classpublic class Mode7
InheritanceMode7 Node
ImplementsIDisplayable, Renderable

This class is very useful to render infinite floor with good rendering quality and perspective correction. This kind of rendering is used in some games like earlier version of MarioKart or other Racing games. It replace successfully a big plane with perspective correction enabled on it.
