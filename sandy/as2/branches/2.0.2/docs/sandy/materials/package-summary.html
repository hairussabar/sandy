Class Summary
Appearance Represents the appearance property of the visible objects.

ColorMaterial Displays a color with on the faces of a 3D shape.
Material The Material class is the base class for all materials.
MaterialManager The MaterialManager class is used to keep track of all materials used in a scene
MaterialType Represents the material types used in Sandy.
VideoMaterial Displays a Flash video ( FLV ) on the faces of a 3D shape.
WireFrameMaterial Displays the faces of a 3D shape as a wire frame.
ZShaderMaterial Displays a kind of Z shading of any object that this material is applied to
