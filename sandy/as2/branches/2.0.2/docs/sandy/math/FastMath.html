Class sandy.math.FastMath

  • sandy.math.FastMath

Description

Fast trigonometry functions using cache table and precalculated data. Based on Michael Kraus implementation.

Field Index

HALF_PI, initialized, PRECISION, PRECISION_DIV_2PI, PRECISION_S, TWO_PI

Method Index

cos(), sin(), tan()

Field Detail

PRECISION

static public PRECISION:Number
The precision of the lookup table.

The bigger this number, the more entries there are in the lookup table, which gives more accurate results.
